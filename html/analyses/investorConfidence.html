Investor Confidence Analysis

Calculate the degree to which private property in an area is backed by financial institutions through mortgages - an indicator of the level of confidence investors have in housing development in the area.

Download the package below to view sample code, detailed tutorials, and project examples that use these analyses.

Download Package

Overview

An investor confidence analysis is completed using loan/mortgage data alongside Census data. It shows the ratio of total mortgaged property to total property value in the study area. A high investor confidence ratio implies that the private sector has high confidence in the financial feasibility of housing development in a certain area, and therefore are more willing to provide loans for the same. A low investor confidence ratio implies the opposite - the private sector is not confident about developing housing in a certain area, and is therefore less likely to help finance construction or ownership.

Data Sources

Source Description
HMDA Data Loan-level data that financial institutions are required to report to the federal government for most mortgage applications. Each record covers a single application and includes the loan type, amount, purpose (purchase, refinance, home improvement), property location down to the census tract, applicant demographics (race, ethnicity, sex, income), and the outcome (originated, denied, withdrawn, etc). The total mortgage value for originated loans represents the numerator in this analysis.
ACS Data Table B25082 reports the aggregate property value for a study area, and is used as the denominator in this analysis.
